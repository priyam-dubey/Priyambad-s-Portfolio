# Personal Portfolio

A clean, responsive personal portfolio website showcasing my journey as a Full-Stack Web Developer, my projects, and my technical skills.

## 🚀 Overview

This portfolio serves as a central hub for my professional identity. It highlights my expertise in building scalable web applications and solving complex algorithmic problems. The site is designed to be performant, user-friendly, and visually appealing.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Tailwind CSS for styling), and JavaScript.
* **Concepts:** Responsive Web Design, UI/UX Principles, and Frontend Optimization.
* **Version Control:** Git & GitHub.

## ✨ Key Features

* **About Me:** A summary of my academic background and passion for software development.
* **Projects:** A showcase of my work, including full-stack applications and AI-integrated tools.
* **Skills:** A categorized list of technical proficiencies (Languages, Frameworks, Databases).
* **Experience:** Highlights from my internships and academic milestones.
* **Contact:** A direct way for recruiters and collaborators to reach out.
